Case Study

PawShop: A Modern Pet E-commerce Experience

A comprehensive case study on designing and developing a pet shop website using only HTML and CSS

Project Overview

Objective

The objective of this project was to create a website using HTML and CSS.
I have decided to create an e-commerce website for a fictional pet shop called PawShop. The idea is to make it visually appealing, easy to navigate, and responsive across different devices.

Key Features & Requirements

  • The website should contain at least three pages.
  • CSS is imported / linked as an external document.
  • The website must have a contact page with a form for users to reach out.
  • The code should include usage of pseudoclasses and selectors.
  • The website should have hover effects.
  • The website should contain CSS animations.
  • The website should have a responsive navigation menu.
  • At least one page needs to be responsive.

The Process

1

Research & Inspiration

Reviewed pet shop and e-commerce sites, plus Pinterest references, to define visual direction and UI patterns.

2

Wireframing & Layout

Created wireframes to map structure, content hierarchy, and key user flows before design.

3

Images

Curated and edited Unsplash images, sourced icons via Iconify, and designed custom SVGs for key feature highlights.

4

Design

Designed the full UI in Figma for mobile and desktop, using an 8pt grid, strong contrast, and a consistent typography system.

5

Development

Built the website with mobile-first approach, with a single desktop breakpoint (968px), ensuring the layout scales cleanly from small screens to larger displays.

6

Testing & Refinement

Tested across screen sizes and browsers, then refined spacing, alignment, and responsiveness for visual consistency.
